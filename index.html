Basic CSS Selectors: Simple and Attribute Selectors

A simple selector is a selector with a single component, such as a single type selector, attribute selector, or pseudo-class, that's not used in combination with or contains any other selector component or combinator. A given element is said to match a simple selector when that simple selector accurately describes the element. Any selector that contains a single selector, attribute selector, pseudo-class, or pseudo-element selector is a simple selector.

Table with 4 Simple Selectors

Syntax / Usage Selector name Description Link to MDN Web Docs
* Universal selector The universal selector, denoted with an asterisk (*), is a special type selector that selects all elements. universal selector
.myClass Class selector The class selector selects all elements that have the given class attribute denoted by the class name prefixed with a period (.) class selector
#myId ID Selector The ID selector selects an element based on the value of its id attribute. The selector is the id prefixed with a "number sign" (U+0023, #). ID selector
div Type selector The type selector selects all elements that have the given node name. For example, div will select all <div> elements and input will match any <input> element. type selector

The CSS attribute selector matches elements based on the element having a given attribute explicitly set, with options for defining an attribute value or substring value match.

Examples of Attribute Selectors

Example Usage Description
a[title] { color: purple; } <a> elements with a title attribute
a[href="https://example.org"] { color: green; } <a> elements with an href matching "https://example.org"
a[href*="example"] { font-size: 2em; } <a> elements with an href containing "example"
a[href$=".org" i] { font-style: italic; } <a> elements with an href ending ".org", case-insensitive
a[class~="logo"] { padding: 2px; } <a> elements whose class attribute contains the word "logo"
